**pull
git init
git remote add origin 주소 - 처음 주소랑 연결하는 거
git pull origin master

**stash
git stash (git pull origin master안될때 이거 치고 pull - 임시로 코드를 저장)
git stash drop stash@{?} - 임시코드 삭제
git stash apply stash@{?} - 임시코드 적용
git stash list - 임시코드 리스트

**push
git status - 바뀐거 확인
git add .
git commit -m "주석"
git push origin master(master - branch이름)

*버전관리
git log - 모든 버전 확인
git reset 6글자 -- hard (6글자는 git log에서 commit나온거 앞에 6글자)

*전버전으로 돌아가서 코딩했는데 github는 최신버전이 존재해서 버전이 맞지 않아 push가 불가능

github를 전버전으로 돌리는 cmd (최신버전들은 다 사라짐)
git push -f origin master

----------------------------------------------------------
branch - 메인 코드 복사본
git branch - branch list 확인
git branch "이름" - 코드 복사본 만드는 거
git checkout "이름" - 그 branch로 바꾸는 거


branch를 수정하고 맘에들어 main에 옮길려 한다.
이때, 기준이 master(git checkout master)
이후 가지고 올 코드를 merge
-git merge "이름"

성공적으로 main에 옮겼으면 branch삭제
git branch -D "이름"
